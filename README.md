# AI Virtual Mouse Pro - Showcase Website

A modern and responsive showcase website for **AI Virtual Mouse Pro**, an AI-powered hand gesture mouse control system.

Coming Soon (Vercel Deployment)

## 🚀 About The Project

This website presents the features, technology stack, and working demonstration of **AI Virtual Mouse Pro**.

The actual application is built using:

- OpenCV
- MediaPipe
- PyAutoGUI
- Python

The core project enables users to control their computer using hand gestures in real time.

## 🎯 Gesture Controls

| Action | Gesture |
|----------|----------|
| Cursor Movement | Index Finger |
| Left Click | Index + Middle Finger |
| Right Click | Index + Middle + Ring Finger |
| Scroll Down | Fist ✊ |
| Scroll Up | Open Palm 🖐️ |
